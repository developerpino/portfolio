# 1984년 1월생 이름은 김민성 성별은 남
평생 택시 운전을 하시던 아버지가 돌아가시기 몇달 전

가누기도 힘든 몸으로 배드민턴을 치러 공원으로 가자고 하셨습니다.

몇번 주고 받으며 벌써 숨이 차 올랐던 아버지는 잘 보라면서 갑자기,

바람이 부는 방향 쪽으로 상당한 높은 각을 내며 위로 치기 시작하였습니다.

공이 떨어지는 방향으로 또 달려가 위로 치고 위로 치는걸 반복하시며 하셨던 말씀이

"민성아 이렇게 하면 혼자서 칠 수 있다. 같이 못쳐줘서 미안해"

입니다.
