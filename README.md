# 자바 전문가 교육 과정 이수
졸업을 앞둔 2009년 여름방학 때,

담당 교수님의 제안으로 한국산업단지공단(kicox) 에서 진행 하는 자바 전문가 교육 과정을 이수 하였습니다.

웹 개발자를 위한 이 교육은 하루 8시간(9시-18시)씩 160 시간의 과정이였고,

자바, oracle, javascript, html, css 웹 전반의 기초부터 시작하여 ibatis, struts, spring 등 framekwork 까지 배울 수 있었습니다.

습득한 기초 지식들을 토대로 조합하여 무언가 무에서 유를 만들어 내고,

막히는 부분에선 document 나 검색을 통하여 배움을 얻어 조금씩 해결을 해 나가는 그 과정에서

이 일이 나에게 보람 이상의 기쁨 나아가 쾌감을 줄 수 있다는 것을 알게 되었습니다.

그렇게 2학기가 시작되면서 마치 고등학교 수학을 선행 학습한 중학생 처럼 에이스가 되어져 있었고,

졸업 전에 취업을 확정하여 웹 개발자로써 일을 시작하게 되었습니다.
